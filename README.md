# Shnekohod
Робот, управляемый с телефона, работающий по принципу шнекохода.
![photo_2025-12-03_15-32-09](https://github.com/user-attachments/assets/3885bc48-531a-4619-8353-3f84bc415e15)

#### Задача:
- Используя материалы и различные технологии изготовить робота на дистанционном управлении для преодоления полосы препятствий
- Робот не должен приводиться в движение колесами

#### Компоненты:
- ESP32-WROOM (+USB Micro)
- Драйвер для двух моторов TB6612
- x3 Аккумулятор Li-ion 18650
- x1 Держатель для 3 аккумуляторов
- x1 LED-лента
- x2 DC-мотор 12В с редуктором
- PCB-шилд
- Силовой ключ (MOSFET)
- Понижающий преобразователь питания
- Резистор
- Светодиод
- Диод Шоттки

#### Описание решения:
Робота приводят в движения два винта архимеда, вращающихся во встречном направлении. Такая конструкция обеспечивает беспрецедентную проходимость в вязкой почве, снеге или песке. На верхней крышке установлена рука-манипулятор.

- Основа робота вырезается из фанеры толщиной 3 мм углекислотным лазером. На неё крепятся все остальные детали
- Неподвижные детали печатаются на 3D-принтере из PLA
- Шнекороторы печатаются на 3D-принтере из PLA и крепятся на подшипники
- Питание обеспечивает блок из трёх аккумуляторов Li-ion 18650
- В качестве основного микроконтроллера используется ESP32-WROOM
- Управление осуществляется через подключение по сети wi-fi.
- Детали для руки-манипулятора также вырезаются из фанеры толщиной 3 или 6 мм углекислотным лазером.

#### Тестирование
В ходе тестирования обнаружена неспособность передвижения робота вперёд по гладкой поверхности.
